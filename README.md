# Neurocysticercosis Disease Classification

A machine learning project focused on classifying Neurocysticercosis disease samples using gene expression data.

## Objective

To develop a classification pipeline that analyzes gene expression patterns and predicts disease classes from genomic data using machine learning techniques.

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* PCA
* Random Forest
* Google Colab

## Project Workflow

1. Data Preprocessing
2. Feature Scaling
3. Dimensionality Reduction (PCA)
4. Model Training
5. Performance Evaluation
